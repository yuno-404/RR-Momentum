## 1. 策略哲學

本策略基於 **William O'Neil（歐乃爾）** 與 **Mark Minervini（乘風破浪的動能操盤手）** 的核心理念：

### 四大信念

| 信念 | 說明 |
|------|------|
| **只買最強的股票** | 只買已經啟動、領先大盤創新高、阻力最小的股票。不撿便宜、不抄底。 |
| **任何市場都能交易** | 不看大盤多空決定能不能買，而是看個股有沒有通過 5 大 Filter。空頭市場中能通過篩選的股票更少，但通過的就是 RS 背離的強勢股，往往是下一輪最大的飆股。 |
| **價量為王** | 不看基本面預測，專注於價格行為（Price Action）、成交量、相對強勢。 |
| **嚴格風控** | 單筆最大虧損 = 總資金的 1%。沒有例外。 |

### 這個策略不做的事

- 不抄底（不買正在下跌的股票）
- 不猜底（不預測大盤何時反轉）
- 不攤平（虧損的部位只會停損，絕不加碼）
- 不因為大盤空頭就放棄尋找強勢股（Filter 本身會自動過濾弱股）

## 3. 第一步：大盤環境判斷 (`/market-regime`)

**大盤環境決定的是「預期能找到多少候選股」，而不是「能不能買」。**

5 大 Filter 本身就是守門員。在空頭市場中，絕大多數股票會自動被 Filter 淘汰（均線死叉、RS 下降、沒有 VCP 收斂），但少數能通過全部篩選的股票，正是展現 RS 背離的強勢股──這些往往是下一輪最大的贏家。

### 三種市場狀態

#### 多頭 ── 候選股多，正常出擊

判斷條件：
- S&P 500、NASDAQ **同時**站上上升中的 50日均線 和 200日均線
- 50MA > 200MA（黃金交叉）
- 市場廣度擴張：創52週新高的股票數 > 創新低的數量
- 各類股輪動健康

**預期**：大量股票通過篩選，正常倉位、正常執行

#### 盤整 / 不確定 ── 候選股變少

判斷條件：
- 指數在 50MA 附近來回震盪
- 廣度訊號混雜
- 沒有明確的類股領頭

**預期**：通過篩選的股票變少，自然會更挑剔

#### 空頭 ── 候選股極少，但 RS 背離股是寶

判斷條件：
- 指數跌破下彎的 50MA
- 50MA < 200MA（死亡交叉或正在收斂）
- 廣度惡化：創新低的股票數急速擴張

**預期**：絕大多數股票被 Filter 淘汰。但如果有股票在這種環境下仍然通過全部 5 個 Filter（RS 高、均線多頭、VCP 成形），**它就是可以買的**。這種股票展現出的 RS 背離說明：大盤跌它不跌，大盤止跌它會率先噴出。

### 空頭中的 RS 背離──核心觀念

你的原始描述：

> 當大盤大跌，強勢股 RS 會與大盤產生背離。
> 指數破底，它抗跌；指數止跌，它噴出。

**這些股票的特徵**：
- 指數創新低 → 它守住前低不破 = RS 背離
- RS 排名在大盤走弱時仍維持 ≥ 85
- 在修正期間建構 VCP 底部型態，消化賣壓
- **如果通過全部 5 大 Filter → 直接可以交易**
- 不需要等大盤轉多頭才能買，因為 Filter 已經確認這檔個股本身處於多頭結構

## 4. 第二步：RS 相對強勢排名

### 計算方式

計算市場中**全部股票**在過去一段時間（通常 6~12 個月）的股價表現，然後給予百分位排名 1~99。

- **RS 90** = 這檔股票的表現贏過全市場 90% 的股票
- **RS 50** = 表現中等
- **RS 10** = 表現墊底

### 關鍵門檻

| 門檻 | 意義 | 出處 |
|------|------|------|
| **RS ≥ 87** | O'Neil 的 87 法則：歷史回測顯示，當一檔股票脫離第一層大底時，若 RS 達到 87 以上，成為超級飆股的機率大幅提升 | William O'Neil |
| **RS ≥ 90** | Minervini 的實證：RS > 90 的強勢股，後續績效顯著優於大盤 | Mark Minervini |
| **RS < 70** | 直接淘汰（大型權值股例外，見下方） | — |

### 大型股 vs 中小型股的差異

| 股票類型 | RS 門檻 | 附加條件 |
|----------|---------|----------|
| 中小型股 | RS ≥ 85 | RS 趨勢必須**向上**，不可與大盤收斂 |
| 大型權值股 | RS ≥ 70 | RS 趨勢必須**向上**，不可與大盤收斂 |

兩者的附加條件相同：**RS 的方向必須向上**。光是 RS 數字高還不夠，趨勢必須是加速或至少持續上升的。差別只在門檻數字（85 vs 70），因為大型股市值大、機構持股多，股價波動天生較小，RS 不會像中小型飆股那麼高。

## 5. 第三步：五大篩選條件

**五個條件必須全部通過。** 有一個沒過就淘汰。

### Filter 1：籌碼淨空 ── 距離一年高點 < 25%

```
               ▲ 52 週高點 $100
               │
    ≤ 25% ─────┤ ← 股價必須在這個區間內
               │
               │    $75 ────── 門檻線
               │
               │
               ▼ 底部
```

**邏輯**：股價越接近歷史高點，上方套牢賣壓越少。距離高點超過 25% 表示上方有大量套牢籌碼需要消化，突破阻力大。

### Filter 2：動能確認 ── 至少從底部上漲 25~30%

```
               ▲
               │    ← 現在的位置
               │
   +25~30% ────┤
               │
               │
               ▼ 底部低點
```

**邏輯**：已經顯著脫離底部區間，代表有真正的機構買盤介入（不是散戶搶反彈的死貓跳）。如果一檔股票還在底部反覆洗盤、漲不上去，代表賣壓太重或沒有主力進場。

### Filter 3：相對強勢 RS 排名

- 中小型股：RS ≥ 85，且 RS 趨勢**向上**（不可與大盤收斂）
- 大型權值股：RS ≥ 70，且 RS 趨勢**向上**（不可與大盤收斂）

**邏輯**：強者恆強。RS 數字高還不夠，方向必須向上。RS 正在下滑或與大盤收斂的股票，即使當前數值仍高，也代表動能正在衰減。

### Filter 4：趨勢保護 ── 長期均線向上

- 50 日均線（50MA）必須在 200 日均線（200MA）**之上**
- 兩條均線都必須**向上傾斜**
- 拒絕沒有方向的橫盤整理

```
        50MA ──────╱  ← 向上傾斜
       200MA ────╱    ← 向上傾斜
                      50MA 在上面 ✓
```

**邏輯**：50MA > 200MA 代表中期趨勢向上，200MA 向上代表長期趨勢向上。我們只在確認的上升趨勢中交易。

### Filter 5：VCP 收斂特徵 ── 波動率收斂型態

突破前必須經歷「震幅遞減 + 量縮打底」的過程。

（詳細說明見下一章節）

## 6. 第四步：VCP 波動率收斂型態

VCP（Volatility Contraction Pattern）是 Minervini 的核心型態。

### 型態長相

```
價格
  ▲
  │     ╲      C1: -25%
  │      ╱╲
  │         ╲     C2: -15%
  │          ╱╲
  │            ╲    C3: -8%
  │             ╱╲
  │              ╲   C4: -4% ← 最後一次收斂（最緊）
  │               ╱──────── ← Pivot Point（突破點）
  │
  └──────────────────────────▶ 時間

成交量
  ▲
  █ █
  █ █ █
  █ █ █ █                          █ ← 突破時放量
  █ █ █ █ █ █                    █ █
  █ █ █ █ █ █ █ █ █ ▄ ▄ ▄ ▄ ▄ █ █ █
  └──────────────────────────────────▶ 時間
      量大                量縮     量增
```

### 關鍵特徵

1. **波動幅度遞減**：每一次回檔的深度比前一次更淺（漏斗狀收斂）
   - 例：第一次回檔 -25% → 第二次 -15% → 第三次 -8% → 第四次 -4%

2. **成交量遞減**：每一次收斂伴隨成交量萎縮
   - 代表賣壓逐漸被吸收，想賣的人已經賣完了

3. **至少 2~3 次收斂**：太少次收斂代表賣壓可能尚未消化完畢

4. **最後一次收斂最緊**：這是最重要的信號
   - 最後一次收斂幅度通常 ≤ 10%
   - 這代表幾乎沒有人願意在這個價位賣出了
   - **這就是最佳的低風險進場點**

### Pivot Point（樞紐突破點）

- 定義：最後一次收斂的**高點**
- 當股價帶量突破這個點時 = 進場信號
- 停損設在最後一次收斂的**低點**下方

### VCP 為什麼有效？

```
第一波回檔：弱手（散戶、短線客）被洗出去
第二波回檔：套牢盤認賠殺出
第三波回檔：最後一批不耐煩的持股者離場
最後收斂：  剩下的都是強手（機構、堅定持股者）
            → 供給枯竭 → 一點點買盤就能推動股價突破
```

## 7. 第五步：倉位計算與風控 (`/position-size`)

### 核心規則

**每筆交易 = 總資金的 12.5%。固定不變。**

不因市場環境、VCP 收斂幅度、交易評分等級而調整。每一筆交易都是 12.5%。

### 公式由來

```
總資金 × 1%（最大容許虧損） ÷ 8%（典型停損） = 12.5%

這就是 12.5% 的來源。
以 8% 停損為基準推導出的固定倉位比例。
```

### 數字範例

```
總資金：$100,000
倉位金額：$100,000 × 12.5% = $12,500
進場價：$50
買入股數：250 股
```

### 停損設定

- 停損 = VCP 最後一次收斂的**低點**下方
- 這是有技術面依據的停損，不是拍腦袋的數字
- 停損幅度決定的是「這筆交易的實際風險比例」，不影響倉位大小

| 停損幅度 | 對總資金的風險 |
|----------|---------------|
| 5%       | 0.625%        |
| 8%（基準）| **1.0%**     |
| 10%      | 1.25%         |
| 12%      | 1.5%          |

### 不攤平、不加碼

- **絕不加碼虧損中的部位**（不攤平）
- **絕不追加獲利中的部位**（不金字塔加碼）
- 一次進場、一個倉位。後續只管理停損和分批出場。

## 8. 第六步：交易管理（R 倍數框架）

### 什麼是 R？

**1R = 你在這筆交易上設定的最大風險金額 = 總資金的 1%**

用 R 來衡量所有交易結果，而不是用金額或百分比，可以讓不同大小的交易具有可比性。

### R 倍數目標價對照表

假設：進場價 $100，停損 $92（8% 停損），1R = $800

| R 倍數 | 股價 | 損益 | 意義 |
|--------|------|------|------|
| **-1R** | $92.00 | -$800 | 停損出場（最大虧損） |
| 0R | $100.00 | $0 | 打平 |
| **+1R** | $108.00 | +$800 | 賺回一倍風險金額 |
| +2R | $116.00 | +$1,600 | 賺回兩倍風險金額 |
| **+3R** | $124.00 | +$2,400 | 賺回三倍風險金額 |
| +5R | $140.00 | +$4,000 | 賺回五倍風險金額 |

### 交易管理三階段

#### 階段一：股價上漲超過 +1R → 停損移至成本價

```
進場 $100 ───────── +1R = $108
                      │
                      ▼
              停損從 $92 移到 $100（成本價）

意義：這筆交易最差的結果 = 沒賺沒賠（免費交易）
這是不可妥協的規則 ── 永遠保護本金。
```

#### 階段二：股價噴出 +2R ~ +3R → 分批停利

```
股價來到 +3R ($124)
         │
         ▼
   賣出 1/3 或 1/2 的持股
   → 收回本金 + 部分利潤
   → 剩下的持股 = "House Money"（莊家的錢）
```

**賣出比例**：你的原始描述是「1/3 OR 1/2」，我保留了兩個選項讓你視情況決定。

#### 階段三：剩餘部位 → JUST HOLD

```
剩下的 1/2 或 2/3 持股：
   → 強勢就一直抱
   → 讓利潤奔跑
   → 直到它不再強勢
```

**出場信號 = 失去相對強勢**：
- RS 開始下滑，或與大盤收斂 → 賣出
- 股票不再是領頭羊（同類股創新高它卻創不了）→ 賣出
- 沒有固定的均線追蹤停損，判斷依據就是「它還強不強」

### 入場前的最低要求

**只做預期報酬至少 3R 的交易。**

```
風險報酬比 = 預期獲利 ÷ 預期虧損 ≥ 3:1

意思是：你願意冒 $1 的風險，但預期可以賺到至少 $3。

如果一個 setup 只看得到 2R 的空間 → 不做。
```

## 9. 第七步：交易日誌 (`/trade-journal`)

### 為什麼要記錄？

沒有紀錄就沒有進步。交易日誌的目的：
1. 客觀記錄每一筆交易（不帶情緒）
2. 用 R 倍數衡量結果（不只看金額）
3. 追蹤紀律：停損有沒有確實執行？出場計畫有沒有遵守？
4. 累積數據計算「期望值」（Expectancy）

### 四種記錄類型

| 類型 | 時機 | 指令範例 |
|------|------|----------|
| **進場記錄** | 買入時 | `/trade-journal 買入 NVDA $120 500股 停損$110` |
| **管理更新** | 調停損或部分出場時 | `/trade-journal NVDA 停損移至成本$120` |
| **出場記錄** | 全部賣出時 | `/trade-journal 賣出 NVDA $155 全部出場` |
| **週報摘要** | 每週回顧 | `/trade-journal 本週摘要` |

### 最重要的指標：期望值（Expectancy）

```
期望值 = (勝率 × 平均獲利R) - (敗率 × 平均虧損R)

範例：
  勝率 40%，平均贏 +3.5R
  敗率 60%，平均輸 -1.0R

  期望值 = (0.4 × 3.5) - (0.6 × 1.0) = 1.4 - 0.6 = +0.8R

  意思：每做一筆交易，平均預期獲利 0.8R
  即使輸的次數比贏的多，只要贏的時候贏得夠大，整體仍然是正期望值。
```

## 10. 第八步：交易審核 (`/trade-review`)

沒有評分系統。5 大 Filter 本身就是篩選機制 ── **全過就做，有一個沒過就不做。**

### 審核清單

| 項目 | 條件 |
|------|------|
| F1 籌碼淨空 | 距離 52 週高點 < 25% |
| F2 動能確認 | 從底部上漲 ≥ 25~30% |
| F3 RS 排名 | 中小型 ≥ 85 且趨勢向上 / 大型 ≥ 70 且趨勢向上 |
| F4 趨勢保護 | 50MA > 200MA，均向上 |
| F5 VCP 收斂 | 波動遞減、量縮、最後收斂最緊 |
| R:R | ≥ 3:1，**低於 3 不做** |

### 判定

- 全部通過 → **TRADE**
- 任何一項沒過 → **NO TRADE**

沒有灰色地帶，沒有打分，沒有「勉強可以」。
